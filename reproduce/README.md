# Reproduce Paper Results

This directory contains two independent reproduction workflows. Each workflow
has its own README with the recommended order of notebooks, expected inputs, and
generated outputs.

## Workflows

| Workflow | Use this for |
| --- | --- |
| [Reproduce scRNASeq Results](<Reproduce scRNASeq Results/README.md>) | Reproducing the scPerturb single-cell benchmark results. |
| [Reproduce Simulated Results](<Reproduce Simulated Results/README.md>) | Reproducing the simulated-data benchmark results. |
